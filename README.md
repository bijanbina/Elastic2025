# Elastic2025
Elastic2025 is a Roundcube Dark Theme based on [Elastic2022](https://github.com/seb1k/Elastic2022) and Elastic

- Design changed to fit android dark theme.
- Better user interface

## Installation
Put all the files in a `elastic2025` subfolder of the the Roundcube's `skins` folder
